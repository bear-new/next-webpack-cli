## a webpack scaffolding

#### 1.Installation scaffolding 
```js npm i next-webpack-cli -g```
#### 2.Init Project
```js next-webpack-cli init react-test```

#### 3.Unload scaffolding
```js npm uninstall next-webpack-cli -g```
